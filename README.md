# 🛍️ Hidden Gem Products: Sentiment-Driven Analysis

### Finding products customers *love* but don’t see enough

This project explores **“hidden gem” products** at JC Penney—items with high customer satisfaction and positive sentiment that fly under the radar due to low visibility.

By combining **structured data (products, ratings)** with **unstructured data (customer reviews)**, I built a custom metric to identify products that deserve more attention.

---

## 🔍 What’s inside

* Data cleaning & normalization (CSV + JSON)
* Merging product metadata, reviews, and user data
* **Sentiment analysis** using VADER (NLP)
* Product-level metrics (rating, sentiment, engagement)
* A custom **Hidden Gem Score** to surface high-potential products
* Visual insights highlighting standout items and brands

---

## 🧠 The idea

A product with a few *very happy* customers can be more valuable than one with many indifferent ones.
The **Hidden Gem Score** balances:

* ⭐ Rating
* 💬 Review sentiment
* 👀 Visibility (number of reviews)

Only ~**2.8%** of products qualified as strong hidden gems.
